# webservice-c-
a service by c#
这是我在线电子合同的后台，使用c#写的。主要是提供android端的后台方案，数据库的设计没有上传，但是代码结构很清晰，主要就是dboperation.cs这个类，它负责对数据库的操作。然后webservice.cs负责展示和调试。弄好之后通过iis进行发布，就可以在android端进行访问了。   <br/>
